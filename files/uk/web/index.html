--- title: Вебтехнології для розробників slug: Web tags: - Landing - Web ---

Відкритий веб надає дивовижні можливості для розробників. Щоб скористатися усіма перевагами цих технологій, потрібно розуміти, як їх застосовувати. Нижче наведено посилання на нашу документацію з вебтехнологій.

Документація для веброзробників

Настанови для веброзробника
Настанови для веброзробника надають корисні поради, які допоможуть вам безпосередньо застосувати вебтехнології для задоволення ваших побажань чи потреб.
Посібники для веброзробників
Набір посібників, що проведуть вас крок за кроком через вивчення API, технологій, чи інших об'ємних областей.
Прогресивні вебзастосунки (PWA)
Прогресивні вебзастосунки - це вебзастосунки, що користуються новітніми можливостями й функціоналом браузерів та стратегією прогресивного покращення, щоб наблизити користувацький досвід використання кросплатформних вебзастосунків до рівня нативних додатків.

Довідник з вебтехнологій

Веб API
Довідник з кожного з окремих API, що разом складають потужні можливості для створення сценаріїв для вебу, включно з DOM і всіма пов'язаними API та інтерфейсами, які можна використовувати для формування вмісту та побудови застосунків у вебі.
HTML
Мова гіпертекстової розмітки - це мова, яка використовується для описання і задання вмісту вебсторінки.
CSS
Каскадні таблиці стилів використовуються для описання зовнішнього вигляду вмісту вебсторінки.
JavaScript
JavaScript - це мова програмування, що застосовується для надання інтерактивності вебсайтам.
SVG
Масштабована векторна графіка дозволяє описувати зображення як набір векторів та форм, що в результаті дозволяє гладко масштабувати ці зображення незалежно від їх оригінального розміру.
MathML
Мова математичної розмітки надає можливість зображати складні математичні рівняння та синтаксис.
Web Components
Вебкомпоненти - це користувацькі елементи, які можна самостійно задавати та повторно використовувати у застосунках.
